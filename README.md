## 📖 Project Overview

This is an intelligent, full-stack Next.js web application designed to revolutionize exam preparation. By uploading standard study materials (PDFs and PPTs), the application leverages the Google Gemini AI engine to transform static text into high-quality, competitive Multiple Choice Question (MCQ) quizzes.

The system is designed not just to test you, but to provide a clear feedback loop on your current knowledge position, helping you identify weak spots before the actual exam.

> **Note:** This repository serves as a **Frontend Demonstration** to showcase UI/UX design, React architecture, and client-side utilities. The proprietary backend AI engine is in the full project folder. For full source code, email me on [muhammad.yazan24@gmail.com]

## 🚀 Key Features

### 1. Smart Document Processing
- **Multi-File Upload:** Seamlessly drag and drop multiple PDF and PPTX slide decks simultaneously.
- **Gemini AI Integration:** Utilizes advanced LLM capabilities to generate contextually accurate and challenging MCQs that mimic real exam patterns.

### 2. Custom Quiz Configuration
- **Quantity Control:** Select exactly how many questions you want to generate (from quick sets to full-length exams).
- **Difficulty Scaling:** Adjust the complexity level (Easy, Medium, Hard) to match your current mastery of the subject.

### 3. Interactive Learning Module
- **User-Friendly UI:** A beautiful, distraction-free interface built for focus.
- **Real-time Feedback:** If a wrong option is selected, the app immediately identifies the correct answer and provides a detailed explanation.
- **Knowledge Assessment:** Track your score in real-time to get an instant pulse on your readiness and "knowledge position" for the subject.

### 4. Dynamic PDF Exporting
The application allows you to take your study material offline with PDF generation:
- **Pre-Quiz Export:** Export a clean, mcq's with answer expanation to practice and learn your topics quickly.
- **Post-Quiz Export:** Export your completed attempt quiz results, including your final score and performance breakdown, and the questions you got wrong with the correct answers and explanations.

## 💻 Frontend Tech Stack & Architecture

- **Framework:** Next.js 14 & React 18
- **AI Engine:** Google Gemini AI (Integration via API)
- **Language:** TypeScript 
- **Styling:** Vanilla CSS3. We intentionally avoided heavy libraries (Tailwind/Bootstrap) to maintain a lightweight footprint and total semantic control.
- **Client Utilities:** `jspdf` for real-time PDF generation.
